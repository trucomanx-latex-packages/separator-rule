\section{Install package}
Put the separator-rule.sty file in any of these locations
\begin{itemize}
\item Put the \texttt{separator-rule.sty} file in the same path of main tex file.
\item Execute the commmand:
\begin{verbatim}
kpsewhich -var-value=TEXMFHOME
\end{verbatim}
and this returns the path of local tex files. By example, if returns 
\begin{verbatim}
/home/username/texmf
\end{verbatim}
then, put the \texttt{separator-rule.sty} file in the directory.
\begin{verbatim}
/home/username/texmf/tex/latex/separator-rule/separator-rule.sty
\end{verbatim}
\end{itemize}



%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\section{Load the package}
To load the package use the next command in the preamble of main tex document.

\begin{verbatim}
\usepackage{separator-rule}
\end{verbatim}
